# ☕ Coffee Shop Sales Analysis

## 📌 Project Overview

This project analyzes coffee shop sales data using Microsoft Excel to uncover business insights, sales trends, and customer purchasing behavior. The analysis helps identify high-performing products, peak sales periods, and revenue patterns to support data-driven business decisions.

---

## 🎯 Objectives

- Analyze overall sales performance
- Identify top-selling products
- Find peak sales hours and days
- Evaluate monthly revenue trends
- Understand customer purchasing behavior
- Build an interactive dashboard

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Dashboard Design

---

## 📊 Key Insights

- Total Sales Analysis
- Revenue Trends
- Product Category Performance
- Best Selling Products
- Peak Business Hours
- Sales by Month
- Interactive Dashboard
